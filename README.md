# Escribanía LP - Gestión de Trámites 08

Plataforma digital diseñada exclusivamente para escribanías, optimizada para la carga, gestión y generación de formularios 08 de transferencia vehicular (automóviles y motocicletas).

## Características Principales

- **Gestión de Trámites**: Interfaz intuitiva para la carga completa de datos de vendedores, compradores y vehículos.
- **Generación de PDF**: Creación automática del formulario 08 oficial listo para imprimir.
- **Sistema de Historial**: Registro centralizado de todos los trámites realizados con capacidad de búsqueda y edición.
- **Borradores Inteligentes**: Permite guardar trámites parciales para ser completados o editados posteriormente.
- **Seguridad**: Acceso protegido mediante autenticación para asegurar la confidencialidad de los datos notariales.
- **Optimización Mobile**: Diseño totalmente responsivo pensado para la operatividad en dispositivos móviles dentro y fuera de la oficina.

Este proyecto utiliza tecnologías modernas como **Next.js**, **Supabase** para la persistencia de datos y **Tailwind CSS** para una interfaz profesional y eficiente.
